Términos de Uso

Condiciones generales para usar RUPE como sistema local de apoyo al reporte y seguimiento de perritos extraviados.

Uso del sistema

RUPE permite registrar mascotas, reportar extravíos, consultar reportes, enviar avistamientos, validar evidencias y administrar catálogos. Su uso debe realizarse con información verídica y respetuosa.

Alcance académico

El sistema es un prototipo de Proyecto Terminal II. No sustituye denuncias, trámites oficiales, servicios veterinarios ni mecanismos institucionales de protección animal.

Responsabilidad del usuario

El dueño de la mascota es responsable de mantener actualizados sus datos, revisar avistamientos, validar coincidencias y cerrar el reporte cuando la mascota sea recuperada.

Responsabilidad del administrador

El administrador local deberá controlar usuarios, catálogos, reportes, bitácora, respaldos y medidas de seguridad. También deberá evitar accesos no autorizados a la base de datos.

Uso prohibido

No se permite registrar información falsa, publicar datos de terceros sin autorización, usar el sistema para fraude, solicitar pagos por recuperación o compartir datos personales fuera de la finalidad del reporte.

Contenido y evidencias

Las fotografías y descripciones se usarán únicamente para identificar mascotas y relacionar reportes con avistamientos. El administrador podrá desactivar contenido duplicado, erróneo o riesgoso.

Disponibilidad local

Al instalarse en una PC, la disponibilidad depende del equipo, energía eléctrica, respaldos y configuración local. Se recomienda crear copias de seguridad periódicas.
